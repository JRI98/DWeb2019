Este trabalho consistiu na realização de duas tarefas:

1ª Criar um transformador em XSL para transformar o documento dos arqueossítios (disponível aqui ) em vários documentos XML, um para cada um dos arqueossítios individuais;

2ª Criar um servidor em Node.js que disponibilize os documentos criados acima.
