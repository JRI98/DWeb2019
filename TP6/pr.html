Abstract:

Este trabalho consistiu no desenvolvimento de um servidor em Node.js capaz de manter a o arquivo sonoro. Para isso, a lista é persistida no servidor sob a forma de um ficheiro no formato JSON , permitindo a consulta de toda a informação e ainda, seguindo a filosofia CRUD:

1- a criação de novos elementos do arquivo sonoro;

2- a consulta de um elemento do arquivo sonoro;

3- a atualização de um elemento do arquivo sonoro,

4- a eliminação de um elemento do arquivo sonoro.
